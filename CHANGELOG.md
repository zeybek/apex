## [0.1.0] - 2026-06-13

### Added

- Three Agent Skills — `apex-design`, `apex-implement`, `apex-review` — each with a `SKILL.md` workflow, conditional `references/`, and `evals/` (trigger and output evals).
- `AGENTS.md` always-on engineering constitution.
- Claude Code and Codex plugin packaging with their marketplace catalogs.
- Dependency-free package and distribution validators (`make validate`), plus official `skills-ref` validation (`make validate-official`).
